# gitinit
gitリポジトリを初期化するスクリプトです。

---
## Description
gitディレクトリで`git init`を実行し、`.gitignore`、`README.md`を作成します。  
`README.md`は空の本文が作成されます。

## How to use
```
$ ./gitinit.py
```

## Requirement
* python2.7 or python3.5
